1. 特征组合18种( A1, A2 ), ( B1, B2, B3 ), ( C1, C2, C3 )
2. 假设空间有48种( A1, A2, * ), ( B1, B2, B3, * ), ( C1, C2, C3, * )
3. 从48种中选择k个，得到$C^k_{48}$个18维向量
4. 对这$C^k_{48}$个18维向量进行去重
5. 发现当k=9时，最多包含9个合取式的析合范式能表示：262144 种假设，已到达$2^{18}$上限
